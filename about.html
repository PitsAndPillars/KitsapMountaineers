Hi, my name is Eric Pitts and this is my first ever website. It is meant to be a mock sports organization website for educational purposes and ths use of CIS 155 at Olympic College.

V1

In v1 I created a few pages and a css stylesheet. Within these pages I set the foundation for the navigation and incorporated some basic styling.

V2

In v2 I laid out how the webpages would be structured and did some graphic design to make the site stand out and be more consistently themed, I also split up my styling into separate layout and styling sheets.

V3

For the third iteration of this site I created a scoreboard using a table, filled out the schedule table using flexboxes/containers, and made the site more mobile friendly by having a scalable navigation, responsive flexcontainer layout, and other minor details.

V4

For the fourth version of this site, I added a form that collects potential players' details to be contacted for tryouts on the team page. I also added an embedded video to share the story of Roberto Clemente, one of the greatest humanitarians in sports history.
